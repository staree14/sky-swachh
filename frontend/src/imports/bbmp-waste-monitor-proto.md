Design a mobile + desktop prototype for an AI-Powered Satellite Waste Monitoring System for Bengaluru (BBMP).

Style: Clean government-tech UI, similar to Google Maps + Swiggy driver dashboard.
Colors: BBMP green + white + grey.
Typography: Modern sans-serif, accessible design.

Screens Needed:

1️⃣ Interactive Map Dashboard

Map of Bengaluru with ward boundaries

Red markers = detected illegal dump sites

Yellow markers = citizen reports pending

Green markers = cleaned sites

Filter panel: ward, severity, date

Click marker → popup with satellite image + citizen photo + status

2️⃣ Citizen Reporting App Screen

One-click geo-tag report

Upload photo

AI auto-fill location + waste type

Track report status timeline

3️⃣ BBMP Officer Dashboard

List of dump alerts with priority score

Assign truck button

SLA timer countdown

Escalation alert UI

4️⃣ Truck Route Optimization Screen

Route map combining regular pickups + illegal dumps

Driver app navigation UI

5️⃣ Analytics Screen

Heatmap of dumping hotspots

Ward-wise performance scorecard

Graphs: detection vs cleanup time

Prototype Requirements

Interactive clickable markers

Navigation between citizen app, officer dashboard, and analytics

Mobile + desktop responsive layout

Use dummy satellite images + Bengaluru landmarks

Goal: Make it look like a real BBMP smart-city monitoring tool.
